# PediZone® - Ayak Sağlığı Ürünleri

Podologların beklentileriyle geliştirilen, klinik olarak test edilmiş ayak sağlığı ürünleri web sitesi.

## 🚀 Özellikler

- ✅ Çok dilli destek (Türkçe, İngilizce, Arapça)
- ✅ SEO optimizasyonu
- ✅ Responsive tasarım
- ✅ Klinik ve profesyonel görünüm
- ✅ Hızlı performans

## 🛠️ Teknolojiler

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## 📦 Kurulum

```bash
# Bağımlılıkları yükle
pnpm install

# Geliştirme sunucusunu başlat
pnpm run dev

# Production build
pnpm run build
```

## 🌐 Netlify Deploy

Bu proje Netlify'da otomatik olarak build edilir:

- **Build Command:** `pnpm run build`
- **Publish Directory:** `dist`
- **Node Version:** 18+
